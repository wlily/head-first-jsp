• JSP Elements and Where in translated Servlet
    • JSP scriptlet
      • <% java code %>
      • scriptlets are copyed to _jspservice() method
    • JSP declaration
      • <%! java code %>
      • declaration becomes the instance variable and method
    • Java expression
      • <%= java code %>
      • expression becomes the input of out.print()
    • JSP directive
      • <%@page attributeName=attributeValue%>
          import
          session
          contentType
          isELIgnored
      • <%@include %>
      • <%@taglib %>
      • directive gives special instructions to the Container at page translation time
    • JSP actions
      • standard action eg: <jsp:include page="ss.jsp"/>
      • non-standard action eg: <c:set var="rate" value="32" />
    • JSP Expression Language(EL)
      • ${something}
      • offer a simpler way to invoke java code
  • JSP Page LifeCycle
    • For only the first request
      • JSP page translation to java
      • JSP page compilation to class
      • load class
      • create instance
      • call jspInit()
    • call _jspService()
    • call jspDestroy()
  • JSP Implicit Objects
    • request------HttpServletRequest
    • response-----HttpServletResponse
    • out----------JspWriter
    • session------HttpSession
    • config-------ServletConfig
    • application--ServletContext
    • pageContext--PageContext
    • page---------Page
    • exception----JspException
